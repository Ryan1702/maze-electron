# 🌀 盲径寻途

一个基于 Electron 的极难迷宫游戏——在迷雾中凭记忆寻找出路。

## 🎮 玩法

- **W/A/S/D** 或 **↑/←/↓/→** 键移动角色
- 从绿色起点 (1, 1) 出发，抵达红色终点 (39, 39) 即为胜利
- 长按方向键可连续移动，自带 120ms 节奏冷却

## 🌫️ 核心机制：视野迷雾

你只能看清周围 **3 格（曼哈顿距离）** 内的迷宫结构。走过的地方立刻被纯黑覆盖，**不保留任何已探索记忆**——你必须凭记忆在 41×41 的庞大迷宫中找到出路。

迷宫在完美迷宫基础上随机打通了约 **5% 的内墙**，制造了环路和冗余路径，进一步增加迷惑性。

## 🚀 快速开始

```bash
# 克隆仓库
git clone git@github.com:Ryan1702/maze-electron.git
cd maze-electron

# 安装依赖
npm install

# 启动游戏
npm start
```

## 📁 项目结构

| 文件 | 说明 |
|------|------|
| `main.js` | Electron 主进程，创建 900×800 窗口 |
| `index.html` | 前端页面，canvas + 信息栏 + 按钮 |
| `style.css` | 深色主题样式 |
| `game.js` | 迷宫生成、渲染、迷雾、移动、计时等核心逻辑 |

## 🛠️ 技术细节

- **迷宫生成**：迭代 DFS（栈实现递归回溯），保证完美迷宫，再随机打通 5% 内墙
- **迷宫尺寸**：41×41（奇数），每格 15px
- **渲染**：Canvas 2D，视野外纯黑遮罩
- **游戏循环**：`requestAnimationFrame` + 120ms 移动冷却
- **安全**：`nodeIntegration: false` + `contextIsolation: true`
