# 🌀 盲径寻途

一个基于 Electron 的极难迷宫游戏——在迷雾中凭记忆寻找出路。

## 🎮 玩法

- **W/A/S/D** 或 **↑/←/↓/→** 键移动角色
- 从绿色起点 (1, 1) 出发，抵达红色终点 (39, 39) 即为胜利
- 长按方向键可连续移动，自带 120ms 节奏冷却

## 🌫️ 核心机制：视野迷雾

你只能看清周围 **3 格（曼哈顿距离）** 内的迷宫结构。走过的地方立刻被纯黑覆盖，**不保留任何已探索记忆**——你必须凭记忆在 41×41 的庞大迷宫中找到出路。

迷宫使用**两步法生成**（v1.0.1 重写）：

 **第一步 — 区块引导散布主路径**：把迷宫分成 4×4 区块，优先选访问最少的区块，让主路径像细长藤蔓贯穿全图（仅占内部总格子的 15% ~ 20%）。每 4 ~ 8 步随机切换"远离/靠近终点"偏好，让正确方向无法预测。

 **第二步 — 从主路径每个点疯狂分叉**：主路径所有格子随机打乱全压入栈，岔路向四面八方爆炸式生长。80% 沿当前方向延伸、20% 强制垂直转弯，死胡同内部也蜿蜒曲折，填满主路径间隙。

 迷宫为**完美迷宫** ，起点到终点仅有一条正确路径，无环路，无冗余通道。

## 🚀 快速开始

```bash
# 克隆仓库
git clone git@github.com:Ryan1702/maze-electron.git
cd maze-electron

# 安装依赖
npm install

# 启动游戏
npm start
```

## 📁 项目结构

| 文件 | 说明 |
|------|------|
| `main.js` | Electron 主进程，创建 900×800 窗口 |
| `index.html` | 前端页面，canvas + 信息栏 + 按钮 |
| `style.css` | 深色主题样式 |
| `game.js` | 两步法迷宫生成（区块引导主路径 + 疯狂分叉）、迷雾渲染、移动、计时等核心逻辑 |

## 🛠️ 技术细节

- **迷宫生成（v1.0.1 两步法）**：第一步区块引导散布主路径（15%~20%），交替偏好远离/靠近终点；第二步从主路径每个点疯狂分叉，80/20 蜿蜒延伸，填满所有剩余格子
- **迷宫性质**：完美迷宫，无环路，任意两点间仅有一条通路
- **迷宫尺寸**：41×41（奇数），每格 15px
- **渲染**：Canvas 2D，视野外纯黑遮罩
- **游戏循环**：`requestAnimationFrame` + 120ms 移动冷却
- **安全**：`nodeIntegration: false` + `contextIsolation: true`
